# KAREN｜银河系争气委员会

一款移动端优先的中文人格测试网页。用户选择 MBTI 后完成 5 道冲突场景题，得到窝囊值、特派员阵容、竖版分享战报、Karen Prompt 与 Agent Soul。

## 运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

V1 使用 `localStorage` 保存最近一次游客战报。登录与云端保存需要后续接入 Supabase。
